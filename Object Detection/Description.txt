The CIFAR-10 dataset consists of 60000 32x32 colour images in 10 classes, with 6000 images per class. There are 50000 training images and 10000 test images.
This project executed using tensorflow library.
It selects any random image from the dataset and matches with the images as per the class and gives the result with the class having more resemblance with the image otherwise print as 'Not in the list'.
